### 后端集成说明
当你准备连接Node.js后端时，只需要在 `submitForm` 函数中替换TODO部分，添加类似这样的代码：

```
const response = await fetch('/api/submit', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify(formData)
})
```
